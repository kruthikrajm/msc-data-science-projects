# Hum & Whistle Song Recognition Model

## Overview
This project develops a machine learning-based song recognition system that identifies songs from user humming and whistling inputs.

## Objectives
- Extract meaningful audio features from humming and whistle recordings.
- Train classification models for song recognition.
- Predict the closest matching song from user input.

## Technologies Used
- Python
- Librosa
- NumPy
- Pandas
- Scikit-learn
- Machine Learning

## Techniques Used
- Audio preprocessing
- MFCC feature extraction
- Feature engineering
- Classification modelling
- Model evaluation

# Large-Scale Taxi Trip Analysis using Apache Spark

## Overview
This project performs large-scale analytics on NYC Yellow Taxi trip data using Apache Spark and PySpark. The objective is to demonstrate distributed data processing and big data analytics techniques.

## Objectives
- Process large-scale datasets efficiently.
- Perform data cleaning and transformation.
- Extract insights from taxi trip patterns.

## Technologies Used
- Apache Spark
- PySpark
- Spark SQL
- GraphFrames
- Python

## Tasks Performed
- Data preprocessing and ETL pipeline development
- Data transformation using Spark DataFrames
- Spark SQL-based analytics
- Trip pattern analysis
- Revenue trend analysis
- Graph-based relationship analysis using GraphFrames

## Key Skills Demonstrated
- Big data processing
- Distributed computing
- Data engineering workflows
- Large-scale analytics

## Key Skills Demonstrated
- Audio data processing
- Feature extraction
- Machine learning classification
- Signal analysis
