# Hum & Whistle Song Recognition Model

## Overview
This project develops a machine learning-based song recognition system that identifies songs from user humming and whistling inputs.

## Objectives
- Extract meaningful audio features from humming and whistle recordings.
- Train classification models for song recognition.
- Predict the closest matching song from user input.

## Technologies Used
- Python
- Librosa
- NumPy
- Pandas
- Scikit-learn
- Machine Learning

## Techniques Used
- Audio preprocessing
- MFCC feature extraction
- Feature engineering
- Classification modelling
- Model evaluation

## Key Skills Demonstrated
- Audio data processing
- Feature extraction
- Machine learning classification
- Signal analysis
